To create a Kafka cluster, apply a Kafka CR in your target namespace
(not {{ .Release.Namespace }}):

  kubectl apply -f - <<EOF
  apiVersion: kafka.strimzi.io/v1beta2
  kind: Kafka
  metadata:
    name: my-cluster
    namespace: <your-namespace>
    annotations:
      strimzi.io/node-pools: enabled
      strimzi.io/kraft: enabled
  spec:
    kafka:
      version: 4.0.0
      metadataVersion: "4.0"
      listeners:
        - name: plain
          port: 9092
          type: internal
          tls: false
      storage:
        type: ephemeral
    entityOperator:
      topicOperator: {}
      userOperator: {}
  EOF

  Note: KRaft mode (above) requires a KafkaNodePool CR — see
  https://strimzi.io/docs/operators/latest/deploying#deploying-kafka-cluster-kraft-str
